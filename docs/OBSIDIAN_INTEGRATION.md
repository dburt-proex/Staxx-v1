# Obsidian Integration

The Workspace page exports the complete learning state as a ZIP vault. Extract the ZIP into an Obsidian vault directory, then open that folder in Obsidian.

The export includes:

- `.obsidian` configuration
- dashboard index
- all fourteen module notes
- generated slide decks, flashcards, podcast scripts, and video manifests
- mastery-review records
- decision ledger entries
- portable application-state JSON

All notes include frontmatter for module number, artifact type, status, version, and provenance where available.

No Obsidian account, API key, or browser secret is required.
